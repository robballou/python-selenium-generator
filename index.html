python-selenium-generator by robballou

A Python module for generating Selenium test code

This is a Python module for creating Selenium test code based on an HTML form. Selenium currently provides a method for finding all elements on a page, but does not provide adequate access to form-specific elements (e.g, the elements within a form). Also, you must currently specify to fill in everything in a piecemeal. This module hopes to help by allowing you to create code for the form you want to test on a page (in case there are multiple forms) and to fill in data based on field types.
